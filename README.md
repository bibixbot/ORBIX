 ORBIX is a Space Debris Prevention Simulator built with GDevelop on Android.

 About
The goal of this project is to simulate satellites orbiting the Earth and explore ways to prevent collisions with space debris. The project is still in development, with the current version focusing on creating a realistic satellite orbit as the foundation for future debris detection and avoidance features.

 Current Progress
- 🌍 Earth and satellite scene created
- 🛰️ Satellite orbits Earth continuously
- ⚙️ Built using GDevelop on Android

 Planned Features
- ☄️ Space debris generation
- 🚨 Collision detection
- 🛡️ Debris avoidance system
- 📊 Score or statistics for prevented collisions

Status
